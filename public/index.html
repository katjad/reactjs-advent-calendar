This advent calendar is made up from explanations of German words, which I have taken from my husband Matin's blog bestandwurstofgerman.net. The first half is therefore not very Christmassy, but there might be more in the second half. Also be prepared for references to football every now and again.

Myself, I made this calendar to learn ReactJS, and also the Node/Express framework, as well as Javascript in general. Almost the whole calendar is just JavaScript (except for static text and image resources of course). You can find the source on GitHub.
